## WGestures 2 是什么？

## 核心功能到底有多强？

### 手势触发方式多到溢出

不只是「按住右键划线」这么简单，WGestures 2 支持的触发方式包括：

- 鼠标右键、中键、侧键（第4/5键）
- 鼠标滚轮
- **屏幕四角触发**（鼠标划到角落就触发，不用按键）
- 屏幕边缘滑动

也就是说，光凭鼠标的各个部位，就能设计出几十种不重叠的触发方式。

### 修饰键组合，手势数量翻倍

这是 WGestures 2 的隐藏绝技。

比如「按住右键向左划」是后退，加上按住 Shift 就变成另一个动作，加上 Ctrl 又是另一个。一个手势方向，搭配不同修饰键，可以衍生出好几个独立动作。手势库容量直接翻几倍。

### 自定义菜单（Windows 独享）

可以把一组相关操作打包成一个菜单，一个手势呼出，再选具体操作。比如选中文字后呼出「搜索菜单」，里面列着百度、谷歌、必应——不用记多个手势，一个手势走天下。

### 脚本支持，高级玩法无上限

支持 AppleScript（macOS）、Cmd 脚本和 Lua 脚本。有编程基础的用户可以用脚本实现几乎任何自动化操作，比如调用 Quicker 动作、控制窗口状态、获取剪贴板内容等。

### 可视化手势轨迹

设置手势时，软件会动态回放你的划动轨迹，一目了然知道这个手势「长什么样」。新手上手友好，不用死记。
